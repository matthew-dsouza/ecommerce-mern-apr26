# ecommerce-mern-apr26

A full-stack MERN e-commerce platform built as a portfolio project.

## Project Goal
Build a real-world mini online store with:
- product listing
- product details
- cart system
- authentication
- checkout flow
- order system
- admin panel
- payment integration
- image upload
- search, filters, and pagination

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Planned Features

### Customer Features
- Browse products
- Search and filter products
- View product details
- Add products to cart
- Persistent cart
- Register and login
- Checkout and place orders
- Leave product reviews and ratings

### Admin Features
- Admin login
- Dashboard
- Product CRUD
- User management
- Order management
- Product image uploads

### Advanced Features
- Stripe or Razorpay integration
- Cloudinary image upload
- Pagination
- Sorting
- Deployment

## Folder Structure
- client/   frontend React app
- server/   backend Express API
- docs/     notes, planning, screenshots
